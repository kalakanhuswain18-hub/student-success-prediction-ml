
# 🎓 Student Success Prediction (ML Project)

## 📌 Overview

This project predicts whether a student will **Pass or Fail** based on various factors like:

* Study Hours
* Attendance
* Past Scores
* Internet Access
* Sleep Hours

It uses a **Machine Learning classification model** built with Scikit-learn.

---

## 🚀 Features

* Data preprocessing (Label Encoding, Scaling)
* Logistic Regression model
* Model evaluation (Confusion Matrix, Classification Report)
* Custom prediction using user input
* Visualization using Seaborn heatmap

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 📊 Model Used

* Logistic Regression (from Scikit-learn)

---

## 📂 Dataset

Dataset contains student-related features:

* StudyHours
* Attendance
* PastScore
* Internet
* SleepHours
* Passed (Target)
